Arena class¶

class Sandbox.Arena(agents: List[Sandbox.Agent.Agent], x_left: float, x_right: float, y_top: float, y_bottom: float)[source]¶

A robot inside an Arena, which will not allow it to escape.¶

A class to represent a rectangular arena, which will confine any agents which are inside of its walls and in its list.

__init__(agents: List[Agent], x_left: float, x_right: float, y_top: float, y_bottom: float)[source]¶

Note: in the current implementation, the code does not check that x_right > x_left and y_top > y_bottom - you have to make sure you get this right yourself.

Parameters

  • agents (List of Agent) – List of agents which are confined by the Arena’s walls. Agents must have a radius attribute for Arena to work.

  • x_left (float) – The x-coordinate of the arena’s left wall.

  • x_right (float) – The x-coordinate of the arena’s right wall.

  • y_top (float) – The y-coordinate of the arena’s top wall.

  • y_bottom (float) – The y-coordinate of the arena’s bottom wall.

step(dt: float, x_move: Optional[float] = None, y_move: Optional[float] = None) → None[source]¶

Step Arena forwards in time. If any Agent s in its agents list have crossed its walls, they will be pushed back inside.

Parameters
